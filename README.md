# NextJS TS Template


## Getting Started

First, download the dependencies:

```bash
npm install
# or
yarn
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Tests:

```bash
npm run test
# or
yarn test
```

Commits:

```bash
npm run commit
# or
yarn commit
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
